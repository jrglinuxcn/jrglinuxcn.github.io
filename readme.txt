## build by honkit

## honkit install
参考: https://honkit.netlify.app/setup

install steps:
1. sudo apt install npm
2. sudo npm install honkit --save-dev


### repository
1. 本仓库记录在gitee, 比较方便pull/push
2. git clone https://gitee.com/jrglinux/jrg-by-jrglinuxcn.github.io.git
3. sudo npx honkit build -- 更新仓库中_book文件夹
3. sudo npx honkit serve -- 本地预览

### github.io更新
1. 拷贝_book文件至jrglinuxcn.github.io仓库, 更新push即可
